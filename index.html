การจัดรูปแบบสตริงใน Python

ในบทความนี้คุณจะได้เรียนรู้วิธีการจัดการข้อความใน Python ด้วยเทคนิคต่างๆ พร้อมตัวอย่างการใช้งานจริง

1. วิธีการจัดรูปแบบสตริงแบบต่างๆ

1.1 % - formatting (แบบดั้งเดิม)


name = "Alice"
age = 30
print("สวัสดี ฉันชื่อ %s และอายุ %d ปี" % (name, age)) 

# การจัดรูปแบบตัวเลข
price = 1234.5678
print("ราคา: %.2f บาท" % price)  # แสดงทศนิยม 2 ตำแหน่ง

Output:

สวัสดี ฉันชื่อ Alice และอายุ 30 ปี
ราคา: 1234.57 บาท

1.2 str.format() (แบบยืดหยุ่น)


# การใช้งานพื้นฐาน
name = "Bob"
price = 99.99
print("สินค้า: {}, ราคา: {:.2f}".format(name, price))

# การระบุตำแหน่ง
print("{1} มีราคา {0:.2f} บาท".format(price, name))

# การใช้ชื่อตัวแปร
print("{product} มีราคา {cost:.2f} บาท".format(
    product=name,
    cost=price
))

Output:

สินค้า: Bob, ราคา: 99.99
Bob มีราคา 99.99 บาท
Bob มีราคา 99.99 บาท

1.3 f-strings (แบบแนะนำ)


quantity = 5
item = "แอปเปิ้ล"
print(f"คุณต้องการซื้อ {quantity} {item}")

# การใช้นิพจน์
price = 20
total = quantity * price
print(f"ราคารวม: {total:,} บาท")

# การเรียกใช้เมธอด
text = "python"
print(f"ข้อความตัวใหญ่: {text.upper()}")

Output:

คุณต้องการซื้อ 5 แอปเปิ้ล
ราคารวม: 100 บาท
ข้อความตัวใหญ่: PYTHON

2. เทคนิคขั้นสูง

2.1 การจัดรูปแบบตัวเลข


number = 1234567.89
print(f"ตัวเลขทั่วไป: {number}")
print(f"ทศนิยม 2 ตำแหน่ง: {number:.2f}")
print(f"คั่นหลักพัน: {number:,.2f}")
print(f"เปอร์เซ็นต์: {number:.2%}")

Output:

ตัวเลขทั่วไป: 1234567.89
ทศนิยม 2 ตำแหน่ง: 1234567.89
คั่นหลักพัน: 1,234,567.89
เปอร์เซ็นต์: 123456789.00%

2.2 การจัดตำแหน่งและความกว้าง


text = "Python"
print(f"|{text:15}|")  # ชิดซ้าย
print(f"|{text:>15}|") # ชิดขวา
print(f"|{text:^15}|") # กึ่งกลาง
print(f"|{text:*^15}|") # เติมด้วยเครื่องหมาย

Output:

|Python         |
|         Python|
|     Python     |
|*****Python*****|
